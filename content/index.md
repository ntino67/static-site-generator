## Selected projects

- **Transvirex Logistics – Microservices ERP**  
  Python/FastAPI + React/TypeScript ERP for a fictional parcel carrier, built around microservices, an API gateway, JWT auth, Postgres-per-service, and a Redis event bus. Owned the Operations service end-to-end and integrated an AI dispatching assistant.  
  M2 team project where I learned that service boundaries are a design problem, not a technical one.

- **EasySave – Backup Tool**  
  C#/.NET backup application with WPF UI, MVVM architecture, AES encryption via external CLI, structured logging, and multithreaded job scheduling.  
  Built as a school deliverable, designed as a maintainable product.

- **Taskforge – Configurable Task Runner**  
  Python CLI task manager supporting JSON, YAML, and TOML configs, with dependency graph execution and structured error handling.  
  Small surface, careful internals — the kind of tool I'd actually use.

- **Static Site Generator**  
  Python tool that parses Markdown and generates the static site you're reading right now.  
  Small, opinionated, and built around my own workflow — easy to extend when I need something new.
